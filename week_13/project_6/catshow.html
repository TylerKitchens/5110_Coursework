Cat SlideShow

  1. Create a variable with var catSlideShow = Vue.extend({...})

    This is shown in catshow.js lines 1-50. Inside of this extend is most of the code given in the zip file with the exception of the template. This template is simple Bootstrap 4.

  2. The next step is registering the component with Vue to use in the HTMML. This was accomplished with Vue.component('catslideshow', slideshow)

    It is important to note that this could have been achieved in one line by combining the step above.

  3. The final step of wiring it to vue was to create a new instance of Vue with new Vue({ el: "#app" })

  4. On the HTML side of things, I gave each component an ID tag then created a button for each 5 cat slideshows.

    <button class="btn btn-primary m-1" onclick="toggle('0')">Aegean</button>

    This called the toggle method

  5. The toggle method took an id of the current element being clicked. It then stores this id so the next time the user clicks another button, the old slideshow will be hidden. I could have hid all of them each time a button was clicked but this way was how my brain was working when originally solving the problem.
